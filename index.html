get-shit-done by hardikr

Small script to configure your hosts file so you don't get distracted during the day.

This is a repository forked from leftnode. You might want to check it out here.

I've added another bash script I picked up on the Hacker News submission of get-shit-done

Dependencies

1. Unix System
2. Python

Install

1. Download/clone.
2. $ chmod +x get-shit-done.py
3. $ sudo cp get-shit-done.py /usr/bin/getshitdone
4. To work: $ sudo get-shit-done work
5. To play: $ sudo get-shit-done play

To edit blocked site list, edit the array site_list.
